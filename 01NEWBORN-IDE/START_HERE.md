## üéØ **YOUR MISSION**

### **Primary Goal**
Build a **modular, scalable form builder** that can:
1. Create dynamic forms via UI
2. Export complete features for RetailMind
3. Embed forms in any application

### **Technology Stack**
- **Backend:** Django 5 + DRF + PostgreSQL
- **Frontend:** React 18 + Vite + MUI 5

- **Forms:** React Hook Form

### **Key Features**

- Dynamic validation (client + server)
- Conditional logic (show/hide)
- Form submission storage
- Export/import JSON
- Package generation
